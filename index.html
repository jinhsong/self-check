전략물자 무허가수출 자가점검

수출신고내역 전략물자 통제 검토

수출신고내역을 판정DB·허가면제·허가DB와 대조하여 전략물자 무허가수출 의심 건을 선별합니다. 모든 처리는 브라우저 내에서만 수행되며 데이터는 외부로 전송되지 않습니다.

1 파일 업로드

엑셀(.xlsx/.xls) 또는 CSV. 시트가 여러 개면 업로드 후 시트를 선택하세요.

2 컬럼 매핑 확인

파일 머리글을 자동 인식했습니다. 틀린 항목은 직접 선택해 주세요. (★ 필수)

3 판정 기준 설정

4 분석 실행

전략물자 선별 → 허가면제/허가 검증 → 무허가 의심건 산출

5 검토 결과

판정 상태 설명 (심각도 순)
  • 무허가·우려국 전략물자 해당·허가/면제 없음 + 도착국이 우려·제재국 → 최우선 확인
  • 무허가(기록없음) 전략물자 해당이나 허가·면제 기록이 전혀 없음
  • 무허가(국가불일치) 통제품목 허가는 있으나 해당 수출국에 대한 허가가 없음(타 국가 허가만 존재)
  • 면제 한도초과 허가면제 대상이나 USD 금액/수량이 설정 한도 초과
  • 허가 기간외 해당 국가 허가는 있으나 수출일이 유효기간을 벗어남
  • 판정미실시·유사모델 DB에 정확히 없으나 유사모델 존재 → 동일통제 여부 확인
  • 판정미실시·미등재 DB에 모델 없음. 무허가 수출은 DB 누락 가능 → 별도 판정 필요
  • USD 산출불가 환율표에 해당 통화/일자가 없어 금액 검증 불가
  • 허가 정상 국가·모델·유효기간 충족
  • 면제 정상 허가면제 대상이며 한도 충족(또는 한도 미설정)
  • 비전략물자 판정DB상 비해당

사내 전략물자 무허가수출 자가점검 보조 도구 · 최종 책임은 수출 담당 부서의 검토에 따릅니다.

사용 안내 (FAQ)

  1. 파일 업로드 — ① 수출신고내역(필수), ② 판정DB(필수), ③ 허가면제 목록, ④ 허가DB, ⑤ 환율표를 올립니다.
  2. 컬럼 매핑 — 머리글을 자동 인식합니다. 빗나간 항목만 직접 선택하세요.
  3. 기준 설정 — 허가면제 USD/수량 한도, 환율 해석 방식, 우려국가 목록을 조정합니다.
  4. 분석 실행 — 전략물자 선별 → 허가면제/허가 검증 → 무허가 의심건을 산출합니다.
  5. 검토 결과 — 심각도 순으로 정렬되며 통제번호·적용환율·허가/면제근거가 함께 표시됩니다. 엑셀로 내려받을 수 있습니다.

모든 처리는 브라우저 안에서만 수행되며, 업로드한 데이터는 외부로 전송되지 않습니다.
