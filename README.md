# 전략물자 무허가수출 자가점검 도구

수출신고내역 엑셀을 **전략물자 판정DB · 허가면제 목록 · 국가별 허가DB**와 대조하여
전략물자 무허가수출 의심 건을 선별하는 단일 HTML 도구입니다.

- **완전 오프라인** — 모든 처리는 브라우저 내부(JavaScript)에서만 수행되며, 수출 데이터는 외부로 전송되지 않습니다.
- 엑셀 파싱은 동봉된 `vendor/xlsx.full.min.js`(SheetJS)를 로컬 참조합니다. 인터넷 연결 없이 동작합니다.

## 사용법

**바로 쓰려면**: `dist_전략물자_점검도구.html` 한 파일만 받아 더블클릭하세요. SheetJS가 안에 포함된
단일 파일이라 `vendor/` 폴더나 인터넷 연결 없이 브라우저에서 바로 열립니다.
(재생성: `node build_standalone.js` — `index.html` + `vendor/`를 묶어 단일 파일 생성)

**개발용**: `index.html`을 엽니다. (`vendor/` 폴더가 같은 위치에 있어야 합니다)

1. 위 파일을 브라우저로 엽니다.
2. **1단계**에서 파일을 업로드합니다. (`.xlsx`/`.xls`/`.csv`, 시트가 여러 개면 선택)
3. **2단계**에서 자동 인식된 컬럼 매핑을 확인하고 틀린 항목만 직접 선택합니다.
4. **3단계**에서 허가면제 USD/수량 한도, 환율 해석 방식, 모델 매칭 방식을 설정합니다.
5. **4단계** `분석 실행` → **5단계**에서 결과 확인 및 엑셀 다운로드.

## 입력 파일과 인식 컬럼

| 파일 | 필수 | 주요 컬럼(자동 인식 키워드 일부) |
|---|---|---|
| ① 수출신고내역 | ★ | 수출신고번호, 모델·규격, 수량, 결제금액, 결제통화, USD금액(있으면), 신고/수리일자, 도착국 |
| ② 전략물자 판정DB | ★ | 모델명, 판정결과(해당/비해당), 판정구분 |
| ③ 허가면제 신고번호 | 선택 | 수출신고번호, (수량한도·금액한도USD 있으면 함께 인식) |
| ④ 국가별 허가DB | 선택 | 국가, 모델명, 유효기간 시작·종료 |
| ⑤ 환율표 | 선택 | 적용일자, 통화, 환율 |

컬럼명은 자유롭게 두어도 키워드로 자동 매핑하며, 빗나간 경우 2단계에서 수동 지정할 수 있습니다.

## 점검 로직

1. **전략물자 선별** — 모델명을 판정DB와 대조.
   - `비해당` → **비전략물자**(점검 대상 아님)
   - DB에 없음 → **판정 미실시** 경고. *무허가로 나간 건은 DB에 누락됐을 수 있으므로 별도 판정 필요* (요청하신 핵심 고려사항)
   - `해당` → 아래 허가/면제 검증으로 진행
2. **허가면제 검증** — 수출신고번호가 면제목록에 있으면, 금액(USD)·수량이 한도 이내인지 확인.
   - 결제금액이 현지화(EUR/INR 등)면 **신고일자 기준 환율**로 USD 환산.
     - 기본(고시환율): `USD = 결제금액 × 통화환율(원) ÷ USD환율(원)` — 환율표에 USD 행이 함께 있어야 교차계산됩니다.
     - 신고서에 USD 금액 컬럼이 있으면 그 값을 우선 사용합니다.
   - 한도 초과 시 **면제 한도초과**.
3. **허가 검증** — 수량은 보지 않고, **국가 + 모델명 + 신고일이 유효기간 이내**인지 확인.
   - 충족 → **허가 정상**, 기간 밖/국가 불일치 → **허가 기간외**.
4. 전략물자인데 허가·면제 어디에도 없으면 → **무허가 의심**(최우선 확인).

## 판정 상태 (심각도 순)

| 상태 | 의미 |
|---|---|
| 무허가·우려국 | 전략물자·허가/면제 없음 + 도착국이 우려·제재국 → **최우선** |
| 무허가(기록없음) | 전략물자 해당이나 허가·면제 기록이 전혀 없음 |
| 무허가(국가불일치) | 통제품목 허가는 있으나 **해당 수출국 허가가 없음**(타 국가 허가만 존재) |
| 면제 한도초과 | 면제 대상이나 USD/수량이 한도 초과 |
| 허가 기간외 | 해당 국가 허가는 있으나 수출일이 유효기간 밖 |
| 판정미실시·유사모델 | DB에 정확히 없으나 유사모델 존재 → 동일통제 여부 확인 |
| 판정미실시·미등재 | DB에 모델 없음(무허가는 DB 누락 가능) |
| USD 산출불가 | 환율표에 통화/일자가 없어 금액 검증 불가 |
| 허가 정상 / 면제 정상 / 비전략물자 | 이상 없음 |

우려국가 목록은 3단계 설정에서 쉼표로 직접 수정할 수 있습니다(기본: 이란·시리아·북한·러시아·벨라루스·쿠바·미얀마·수단).
결과 표·엑셀에는 **통제번호·판정구분(자가/전문)·적용환율·HS부호·허가/면제근거(허가번호·면제사유)** 가 함께 출력됩니다.

## 고위험 품목 키워드 경고

판정 상태와 **별개로**, 자재코드·거래품명·모델규격 텍스트(+HS Code)에서 고위험 품목 키워드를
탐지해 행별 🚨 태그·요약 카드·상단 경고 배너로 알립니다. 탐지 범주(18종):

3차원측정기 · 열화상 카메라 · 공작기계(CNC/Milling/EDM 등) · 클린룸 · 오실로스코프 ·
주파수변환기 · 신호분석기 · 신호발생기 · 회로망분석기 · 3D 프린터 · 워크스테이션 · 서버 ·
이더넷 스위치 · 네트워크 기지국 → 기지국 모델코드(SLS-/SFG-/SMM-) → 기지국 HS(8517792000 또는 851762로 시작)
(기지국 3단계는 우선순위 적용) · GPU(Nvidia/H100/A100 등, HS 84·85 한정) · FPGA(HS 84·85 한정)

매칭 규칙: 대소문자 무시, 단어경계 적용("L4"가 "L40"에, "Server"가 "Observer"에 안 걸림),
영문 복수형(-s) 허용, 공백·하이픈·붙여쓰기 표기 차이 허용. 규칙은 `index.html`의
`HIGHRISK_RULES` 상수에서 직접 수정할 수 있습니다.

## 샘플 데이터 / 데모 결과

`samples/` 폴더에 가상의 정밀계측·전자부품 수출사 시나리오(수출신고 15건)가 들어 있습니다.
그대로 ①~⑤에 업로드하고 `허가면제 USD 한도 = 1000`으로 분석하면 8가지 상태가 모두 재현됩니다.

| 파일 | 용도 |
|---|---|
| `수출신고내역_2026.xlsx` | ① 점검 대상 15건 |
| `전략물자_판정DB.xlsx` | ② 자가/전문판정 결과 |
| `허가면제_목록.xlsx` | ③ 허가면제 신고번호 |
| `수출허가_DB.xlsx` | ④ 국가별 허가 |
| `주간_고시환율.xlsx` | ⑤ 환율(원/통화) |
| `점검결과_예시.xlsx` | 위 입력에 대한 도구 산출 결과(예시 출력) |

데모 요약(19건): **무허가·우려국 1** (SDR-100 이란), **무허가(국가불일치) 2** (HPA-9000 중국·ENC-55 미국),
**면제 한도초과 2**, **허가 기간외 1** (SCM-7200 일본 2027), **판정미실시·미등재 4** (NEWX-1 등),
허가 정상 4 / 면제 정상 1 / 비전략물자 4. **고위험 키워드 4건**: 공작기계(CNC Milling·판정 비해당인데도 경고),
서버+GPU(AI Server·H100·HS 84), 기지국 모델코드(SLS-4400B), 기지국 HS(TTM-300·8517.62).

재생성: `cd samples && npm i xlsx && node generate_samples.js`

> 자동 매칭/환율 해석은 실제 파일 양식에 맞춰 조정이 필요할 수 있습니다.
> 실제 수출신고내역을 첨부해 주시면 컬럼 인식·통화·국가표기 등을 맞춰 보정합니다.
