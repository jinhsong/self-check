# 전략물자 무허가수출 자가점검 도구

수출신고내역 엑셀을 **전략물자 판정DB · 허가면제 목록 · 국가별 허가DB**와 대조하여
전략물자 무허가수출 의심 건을 선별하는 단일 HTML 도구입니다.

- **완전 오프라인** — 모든 처리는 브라우저 내부(JavaScript)에서만 수행되며, 수출 데이터는 외부로 전송되지 않습니다.
- 엑셀 파싱은 동봉된 `vendor/xlsx.full.min.js`(SheetJS)를 로컬 참조합니다. 인터넷 연결 없이 동작합니다.

## 사용법

1. `index.html`을 브라우저로 엽니다. (`vendor/` 폴더가 같은 위치에 있어야 합니다)
2. **1단계**에서 파일을 업로드합니다. (`.xlsx`/`.xls`/`.csv`, 시트가 여러 개면 선택)
3. **2단계**에서 자동 인식된 컬럼 매핑을 확인하고 틀린 항목만 직접 선택합니다.
4. **3단계**에서 허가면제 USD/수량 한도, 환율 해석 방식, 모델 매칭 방식을 설정합니다.
5. **4단계** `분석 실행` → **5단계**에서 결과 확인 및 엑셀 다운로드.

## 입력 파일과 인식 컬럼

| 파일 | 필수 | 주요 컬럼(자동 인식 키워드 일부) |
|---|---|---|
| ① 수출신고내역 | ★ | 수출신고번호, 모델·규격, 수량, 결제금액, 결제통화, USD금액(있으면), 신고/수리일자, 도착국 |
| ② 전략물자 판정DB | ★ | 모델명, 판정결과(해당/비해당), 판정구분 |
| ③ 허가면제 신고번호 | 선택 | 수출신고번호, (수량한도·금액한도USD 있으면 함께 인식) |
| ④ 국가별 허가DB | 선택 | 국가, 모델명, 유효기간 시작·종료 |
| ⑤ 환율표 | 선택 | 적용일자, 통화, 환율 |

컬럼명은 자유롭게 두어도 키워드로 자동 매핑하며, 빗나간 경우 2단계에서 수동 지정할 수 있습니다.

## 점검 로직

1. **전략물자 선별** — 모델명을 판정DB와 대조.
   - `비해당` → **비전략물자**(점검 대상 아님)
   - DB에 없음 → **판정 미실시** 경고. *무허가로 나간 건은 DB에 누락됐을 수 있으므로 별도 판정 필요* (요청하신 핵심 고려사항)
   - `해당` → 아래 허가/면제 검증으로 진행
2. **허가면제 검증** — 수출신고번호가 면제목록에 있으면, 금액(USD)·수량이 한도 이내인지 확인.
   - 결제금액이 현지화(EUR/INR 등)면 **신고일자 기준 환율**로 USD 환산.
     - 기본(고시환율): `USD = 결제금액 × 통화환율(원) ÷ USD환율(원)` — 환율표에 USD 행이 함께 있어야 교차계산됩니다.
     - 신고서에 USD 금액 컬럼이 있으면 그 값을 우선 사용합니다.
   - 한도 초과 시 **면제 한도초과**.
3. **허가 검증** — 수량은 보지 않고, **국가 + 모델명 + 신고일이 유효기간 이내**인지 확인.
   - 충족 → **허가 정상**, 기간 밖/국가 불일치 → **허가 기간외**.
4. 전략물자인데 허가·면제 어디에도 없으면 → **무허가 의심**(최우선 확인).

## 판정 상태

| 상태 | 의미 |
|---|---|
| 무허가 의심 | 전략물자이나 허가·면제 기록 없음 |
| 판정 미실시 | 판정DB에 모델 없음(누락 가능) |
| 면제 한도초과 | 면제 대상이나 USD/수량이 한도 초과 |
| 허가 기간외 | 모델 허가는 있으나 수출일이 유효기간 밖 |
| USD 산출불가 | 환율표에 통화/일자가 없어 금액 검증 불가 |
| 허가 정상 / 면제 정상 / 비전략물자 | 이상 없음 |

## 샘플 데이터 / 데모 결과

`samples/` 폴더에 가상의 정밀계측·전자부품 수출사 시나리오(수출신고 15건)가 들어 있습니다.
그대로 ①~⑤에 업로드하고 `허가면제 USD 한도 = 1000`으로 분석하면 8가지 상태가 모두 재현됩니다.

| 파일 | 용도 |
|---|---|
| `수출신고내역_2026.xlsx` | ① 점검 대상 15건 |
| `전략물자_판정DB.xlsx` | ② 자가/전문판정 결과 |
| `허가면제_목록.xlsx` | ③ 허가면제 신고번호 |
| `수출허가_DB.xlsx` | ④ 국가별 허가 |
| `주간_고시환율.xlsx` | ⑤ 환율(원/통화) |
| `점검결과_예시.xlsx` | 위 입력에 대한 도구 산출 결과(예시 출력) |

데모 요약: **무허가 의심 3** (HPA-9000 중국·ENC-55 미국·SDR-100 이란), **판정 미실시 1** (NEWX-1),
**면제 한도초과 2**, **허가 기간외 1** (SCM-7200 일본 2027), 허가 정상 4 / 면제 정상 1 / 비전략물자 3.

재생성: `cd samples && npm i xlsx && node generate_samples.js`

> 자동 매칭/환율 해석은 실제 파일 양식에 맞춰 조정이 필요할 수 있습니다.
> 실제 수출신고내역을 첨부해 주시면 컬럼 인식·통화·국가표기 등을 맞춰 보정합니다.
